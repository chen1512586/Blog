#  **基于Vue+Spring boot搭建的博客项目** 

#### 介绍
个人博客

#### 软件架构
Blog和blog_backstage为前端项目，其中Blog是前台，blog_backtage是后台
biyesheji为后端项目

#### 所用技术

前端： "vue" + "vuex" + "vue-router" + "axios" + "element" + "echarts"
后端： "SpringBoot" +  "Mysql"
